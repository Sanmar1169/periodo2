CSS

¿Que es CSS?

Es un lenguaje que nos permite hacer paginas web atractivas.

¿Qué es una referencia por palabra clave?

Esta Referencia CSS muestra la sintáxis básica de una regla CSS; lista todas las propiedades estándares CSS, pseudo-classes y pseudo-elementos, reglas-at, unidades, y selectores, todos juntos en orden alfabético, así como los selectores por tipo; y le permitirá acceso rápido a la información detallada de cada uno de ellos. No solo lista las propiedades de CSS 1 y CSS 2.1, sino que también es una referencia de CSS3 que enlaza cualquier propiedad y concepto de CSS3 estandarizado, o ya establecido. También incluye una breve referencia DOM-CSS / CSSOM.

Ejemplo:

div {color: red; } h1 {color: initial; }

¿Qué es un referencia por ID

En un documento HTML, los selectores de ID de CSS buscan un elemento basado en el contenido del atributo id . El atributo ID del elemento seleccionado debe coincidir exactamente con el valor dado en el selector.

Ejemplo:

#identified { background-color: skyblue; }

¿Qué es una referencia por clase?

El atributo class está definido como una lista de elementos separados por espacio, y uno de esos elementos debe coincidir exactamente con el nombre de clase dado en el selector.

Ejemplo:

span.classy { background-color: DodgerBlue; }
